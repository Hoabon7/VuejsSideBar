## Install Project

```
git clone git@github.com:regenrek/vue-sidebar-menu-example.git
```

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```
